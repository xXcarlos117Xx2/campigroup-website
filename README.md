# Campigroup Website

Pagina web diseñada para dar información sobre juegos siendo ejecutados en el servidor oficial de [Campigroup](http://www.campigroup.es).
Es una [landing page](https://es.wikipedia.org/wiki/Página_de_aterrizaje) preparada con el conocimiento adquirido en la academia de [4Geeks](https://4geeksacademy.com/es/inicio?lang=es), tanto frontend como backend han sido desarrollados por mi ([Carlos Atanes](https://github.com/xXcarlos117Xx2))

![Estado del Proyecto](https://img.shields.io/badge/estado-en%20desarrollo-yellow)

## Instrucciones de Configuración Local

1. Clona el repositorio.
2. Instala las dependencias con `npm install`.
3. Ejecuta la aplicación con `npm start`.

o ejecuta un codespace directamente desde la rama `master`.

## Estructura de Carpetas y Archivos

- `src`: Archivos del codigo sin compilar.
	- `img`: Imágenes usadas en la web.
	- `js`: Codigo javascript de la web.
		- `component`: Componentes js que conforman el esqueleto de la web.
		- `store`: Componente que controla los estados y datos globales de la web.
		- `views`: Las distintas vistas web o "partes" de la website.
	- `styles`: Codigo CSS de estilado.
- `template.html`: Codigo HTML básico para la carga de React, Bootstrap, Font-Awesome entre otras...
- `public`:
